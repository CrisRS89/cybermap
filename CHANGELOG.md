# Changelog

## v0.1.0

Initial distributable MVP release.

### Added

- FastAPI backend.
- Next.js frontend.
- Settings UI.
- Settings API.
- Frontend-backend settings synchronization.
- Local JSON settings persistence.
- Hardened settings payload validation.
- Automated local validation script.
- Fresh clone validation script.
- Product CLI entrypoint with `./bin/cybermap`.
- Setup, dev, test and validation scripts.
- Manual E2E validation documentation.
- Fresh clone setup documentation.
- Repository access documentation.

### Validated

- Backend tests pass.
- Frontend lint passes.
- Frontend tests pass.
- Frontend production build passes.
- Manual Settings E2E flow passes.
- Fresh clone validation passes.
- `main` published to GitHub.
- `v0.1.0` tag published to GitHub.

### Limitations

This release does not include:

- real AI provider execution
- real MCP integration
- real agent execution
- external connectors
- multi-user authentication
- production database
- secret storage
- Docker deployment

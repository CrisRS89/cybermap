# CyberMap Architecture

## Objetivo

Definir la arquitectura técnica de CyberMap.

## Capas principales

- Frontend Web
- Backend API
- Core Domain
- AI Provider Gateway
- Agent Hub
- MCP Client Layer
- Connector Hub
- Scanner Worker
- Agent Worker
- Reports Engine
- Audit Logs

## Principios

- Separación de responsabilidades
- Seguridad por diseño
- Mínimo privilegio
- Configuración por entorno
- Auditoría de acciones sensibles

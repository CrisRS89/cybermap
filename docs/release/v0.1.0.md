## Instalacion recomendada

Para repositorio publico:

    git clone https://github.com/CrisRS89/cybermap.git
    cd cybermap
    ./bin/cybermap install
    ./bin/cybermap dev

Para repositorio privado:

    git clone git@github.com:CrisRS89/cybermap.git
    cd cybermap
    ./bin/cybermap install
    ./bin/cybermap dev

## Comandos principales

    ./bin/cybermap help
    ./bin/cybermap install
    ./bin/cybermap validate
    ./bin/cybermap test
    ./bin/cybermap dev

## Validacion tecnica

La validacion principal es:

    ./bin/cybermap validate

## Limites

Esta release no incluye todavia:

- IA real
- MCP real
- ejecucion real de agentes
- conectores externos
- secretos reales
- autenticacion multiusuario
- base de datos productiva
- Docker
- CI/CD remoto

## Criterio de aceptacion v0.1.0

La release es aceptada si:

1. el repo puede clonarse
2. `./bin/cybermap install` termina correctamente
3. `./bin/cybermap validate` pasa completo
4. `./bin/cybermap dev` levanta backend y frontend
5. la UI carga en `http://localhost:3000`
6. la API responde en `http://localhost:8000`
